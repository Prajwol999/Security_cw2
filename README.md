
Dayalu Haath  is a secure and comprehensive web-based platform designed to bridge the gap between blood donors and recipients. Built using a modern technical stack comprising a React frontend and a Node.js/Express backend, the application facilitates real-time coordination through features like instant chat, user dashboards, and automated notifications. Security is a core priority, evidenced by the implementation of Two-Factor Authentication (2FA) and Google OAuth, ensuring a safe environment for users to manage donation requests and personal profiles while streamlining the critical process of medical assistance.
